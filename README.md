# MemoTest
## Requisitos

Antes de comenzar, asegurate de tener los siguientes requisitos instalados en tu sistema:

- **Node.js** (versión 20.17.0 o superior)  
  Puedes descargarlo desde [acá](https://nodejs.org/).

- **npm** o **yarn** (instalados con Node.js)  
  - Verifica que estén instalados ejecutando:
    ```bash
    node -v
    npm -v
    ```

- **Git** (opcional, pero recomendado para manejar el repositorio).

## Pasos para levantar el juego

Sigue estos pasos para ejecutar el juego en tu máquina local:

1. **Clona el repositorio**  
   Usa el siguiente comando para clonar el proyecto desde GitHub:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   cd <NOMBRE_DEL_REPOSITORIO>
   ```

2. **Instala las dependencias**  
    Ejecuta el siguiente comando para instalar las dependencias necesarias:

    ```bash
    npm install
    ```

3. **Inicia el servidor de desarrollo**  
    Una vez instaladas las dependencias, inicia el servidor de desarrollo con el siguiente comando:

    ```bash
    npm run dev
    ```

4. **Abri la aplicación en el navegador**  
    Cuando el servidor esté en funcionamiento, abre el navegador y accede a la siguiente dirección:

    ```bash
    http://localhost:5173
    ```
